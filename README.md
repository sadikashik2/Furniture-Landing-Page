### Furniture Landing Page

A **modern, responsive furniture landing page** built with **React**, **Vite**, and **Tailwind CSS**.

-------
**Live Demo:**  https://furniture-landing-page-zeta.vercel.app/

### Features

1. **Responsive Design**
   - Works on **desktop, tablet, and mobile**.

2. **Dark/Light Theme**

3. **Shopping Cart**
   - Items saves into local storage.
   - Add items to cart.
   - Remove items from cart.
   - Increase or decrease item quantity.
   - Clear all items with one click.

4. **Mobile Menu Slider**

-------

### Technologies Used

- **React**
- **Vite**
- **Tailwind CSS v4**
- **React Router** 
- **React Icons** – Cart, menu, and theme icons  
- **React Toastify** – Toast notifications for cart actions
